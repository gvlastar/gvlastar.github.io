Connecting to campus WiFi
Parents & Guests
  • Connect to CMU Guest
  • Go to any web page
  • Enter valid email and access code in redirected login page
Your IOS device
  • Find CMU-SECURE in your Wi-Fi list.
  • Enter Username: Andrew userID
  • Enter Password: Andrew password
  • If prompted, verify and accept the CMU wireless certificate.
Your Android device
  • Find CMU-SECURE in your Wi-Fi list.
  • EAP method: PEAP
  • Phase 2 authentication: MSCHAPV2
  • CA certificate: Use System Certificate
  • Domain: wireless.cmu.edu
  • User certificate: Unspecified
  • Identity: Andrew userID
  • Password: Andrew password
Your Laptop
Enter andrew ID and password for CMU-SECURE
Printing: things to know
How to Print
  1. Download software from CMU website
  2. Select printer (image)
  3. Swipe ID at locations
Where to Print
    8.5"x11" or 11"x17"
  • Academic buildings: Gates L3/L5, Wean L3/L5, CFA L3, Tepper L2/L3, Baker L1, Hamerschlag A/L1, Doherty L1, INI B1/B2
  • Administrative buildings: Hunt Library, Cohon University Center L1, Cyert Hall L1
  • Residential buildings: Morewood, Mudge, West Wing, The Hill, Donner, Rez on Fifth
  • See: Tartan Ink
Good to know
  • Print up to 800 sheets free each semester ($0.05-$1.50 each, $40 quota)
  • Additional quota - PlaidCash (See: The HUB)
  • Can check Printer Status on CMU Website
